API [Яндекс Расписаний](https://yandex.ru/dev/rasp/doc/concepts/about.html) — инструмент программного доступа к данным сервиса о маршрутах и времени движения транспортных средств.

API организован в соответствии с принципами REST. Взаимодействие с API осуществляется по протоколу HTTPS посредством GET-запросов. Сервис возвращает данные в форматах: JSON, XML

Необходимо с помощью Postman протестировать работу следующих методов:

- расписание рейсов между выбранными станциями (дописать станции);
- расписание рейсов, отправляющихся от указанной станции (дописать станцию);
- маршрут с указанием времени в пути и всех остановок (дописать маршрут)
- список ближайших к заданной точке станций (дописать точку).
